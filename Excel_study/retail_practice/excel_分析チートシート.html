① 集計関数

SUM ― 単純合計

=SUM(J2:J1001)          # J列全合計
=SUM(J:J)               # 列全体（ヘッダ行は自動スキップ）

SUMIF ― 条件1つの合計

=SUMIF( 条件範囲, 条件, 合計範囲 )
=SUMIF(F:F, "家電", J:J)                # カテゴリ=家電の売上合計
=SUMIF(J:J, ">10000", J:J)             # 10,000円超の合計
=SUMIF(E:E, A2, J:J)                   # A2の地域に一致する合計（セル参照）
条件が1つならSUMIF。条件に演算子を使う場合は ">10000" のように文字列で囲む。

SUMIFS ― 複数条件の合計

=SUMIFS( 合計範囲, 条件範囲1, 条件1, 条件範囲2, 条件2, … )
※ SUMIFと引数順が違う！ 合計範囲が最初
=SUMIFS(J:J, D:D,"ゴールド", E:E,"東京")
# 会員ランク=ゴールド かつ 地域=東京 の売上合計

=SUMIFS(J:J, E:E,$A2, F:F,B$1)
# クロス集計用：行方向に地域($A2)、列方向にカテゴリ(B$1)で参照

=SUMIFS(J:J, B:B,">="&DATE(2025,4,1), B:B,"<"&DATE(2025,7,1))
# 日付範囲条件：2025年Q2（4〜6月）の合計

SUMPRODUCT ― 配列演算で集計（多用途）

=SUMPRODUCT( 配列1, 配列2, … )
各配列を要素ごとに掛け算して合計。条件は (条件式)*1 で0/1に変換
=SUMPRODUCT(H2:H1001, I2:I1001)
# 単価×数量の合計（小計列なしで総売上）

=SUMPRODUCT((MONTH(B2:B1001)=4) * J2:J1001)
# 4月の売上合計（SUMIFS不可のMONTH条件をここで使う）

=SUMPRODUCT((E2:E1001="東京") * (F2:F1001="家電") * J2:J1001)
# AND条件は * （乗算）、OR条件は + （加算）で表現
範囲は同じ行数にそろえること。列全体指定（J:J）はSUMPRODUCTでは使わない（重い）。

② カウント関数

関数用途例
COUNT数値セルの個数=COUNT(J2:J1001)
COUNTA空白以外の個数=COUNTA(A2:A1001)
COUNTBLANK空白セルの個数=COUNTBLANK(D2:D1001)
COUNTIF条件1つでカウント=COUNTIF(E:E,"東京")
COUNTIFS複数条件でカウント=COUNTIFS(E:E,"東京",D:D,"ゴールド")
=COUNTIF(J:J, ">50000")            # 5万円超の取引件数
=COUNTIFS(F:F,"家電", K:K,"クレジットカード")
# 家電 かつ クレジットカード の件数

=SUMPRODUCT((MONTH(B2:B1001)=1)*1)
# 1月の取引件数（COUNTIFSは日付のMONTH条件が不可→SUMPRODUCT）

③ 平均関数

=AVERAGEIF( 条件範囲, 条件, 平均範囲 )
=AVERAGEIFS( 平均範囲, 条件範囲1, 条件1, … )
=AVERAGEIF(D:D, "ゴールド", J:J)
# ゴールド会員の平均購入金額

=AVERAGEIFS(J:J, E:E,"東京", F:F,"家電")
# 東京×家電 の平均購入金額

=IFERROR(AVERAGEIF(D:D, A2, J:J), 0)
# 該当なし（#DIV/0!）を0に置換
AVERAGEIFS は SUMIFS と引数順が同じ（平均範囲が最初）。

④ 参照関数

VLOOKUP ― 縦方向検索（古典的手法）

=VLOOKUP( 検索値, テーブル範囲, 列番号, FALSE )
列番号: テーブルの左端列=1 として数える ／ 第4引数は必ずFALSE（完全一致）
=VLOOKUP(A2, '2_商品マスタ'!$A:$D, 2, FALSE)
# A2の商品コードで商品名（2列目）を検索

=VLOOKUP(A2, '2_商品マスタ'!$A:$D, 4, FALSE)
# 標準単価（4列目）を検索
列挿入で列番号がズレる。マスタを変更した場合は必ず確認。

365/2021 XLOOKUP ― 現代的な参照

=XLOOKUP( 検索値, 検索範囲, 戻り範囲, [見つからない場合], [一致モード] )
=XLOOKUP(A2, '2_商品マスタ'!$A:$A, '2_商品マスタ'!$B:$B, "該当なし")
# 商品コードで商品名を検索、未登録は「該当なし」

=XLOOKUP(A2, '2_商品マスタ'!$A:$A, '2_商品マスタ'!$B:$D)
# 複数列まとめて返すことも可能（商品名・カテゴリ・単価）
VLOOKUPより列挿入に強く、未検索時の表示も指定しやすい。

INDEX + MATCH ― 柔軟な2次元検索

=INDEX( 戻り範囲, MATCH( 検索値, 検索範囲, 0 ) )
=INDEX('1_売上データ'!A:A, MATCH(B2,'1_売上データ'!J:J, 0))
# 売上金額がB2と一致する行の取引IDを返す

=INDEX('2_商品マスタ'!B:B, MATCH("ダンベル",'2_商品マスタ'!B:B, 0))
# 商品名「ダンベル」の行を特定してB列を返す
MATCH の第3引数は 0（完全一致）が鉄則。1/-1は並び順前提になる。

⑤ 日付関数

関数構文例・結果
YEAR=YEAR(B2)2025
MONTH=MONTH(B2)1〜12
DAY=DAY(B2)1〜31
DATE=DATE(2025,4,1)2025/4/1 のシリアル値
EOMONTH=EOMONTH(B2, 0)B2の月末日（1=翌月末）
WEEKDAY=WEEKDAY(B2, 2)1=月〜7=日（第2引数2）
TEXT=TEXT(B2,"yyyy/mm")"2025/01"（文字列）
DATEDIF=DATEDIF(A2,TODAY(),"M")経過月数
NETWORKDAYS=NETWORKDAYS(A2,B2,祝日)営業日数
WORKDAY=WORKDAY(A2,10,祝日)10営業日後

月別・四半期別集計のパターン

# 月列を別途作ってSUMIFS（高速）
# 作業列: =MONTH(B2) → 下方向コピー → その列でSUMIFS
=SUMIFS(J:J, L:L, 4)   # L列に月番号がある場合

# 作業列なしでSUMPRODUCT（遅いが1セルで完結）
=SUMPRODUCT((MONTH(B$2:B$1001)=A2)*J$2:J$1001)

# 四半期判定（INT関数で1〜4に変換）
=INT((MONTH(B2)-1)/3)+1   # 1月→1Q, 4月→2Q ...

⑥ 統計関数

基本統計

関数意味
AVERAGE算術平均
MEDIAN中央値（外れ値に強い）
MODE最頻値
MAX / MIN最大・最小
MAXIFS / MINIFS条件付き最大・最小 (365)

散布・分位

関数意味
STDEV標本標準偏差
STDEVP母標準偏差
VAR / VARP分散
PERCENTILEパーセンタイル(0〜1)
QUARTILE四分位(0=min,1=Q1,…)
=MEDIAN(J2:J1001)                          # 中央値
=STDEV(J2:J1001)                           # 標準偏差
=PERCENTILE(J2:J1001, 0.9)                 # 上位10%の閾値
=PERCENTILE(J2:J1001,0.75)-PERCENTILE(J2:J1001,0.25)
# IQR（四分位範囲）＝外れ値検知に使う

=MINIFS(J:J, E:E,"東京")                   # 東京の最小売上（Excel365）

⑦ ランキング

関数用途例
LARGE(範囲,n)n番目に大きい値=LARGE(J:J,1) → 最大値
SMALL(範囲,n)n番目に小さい値=SMALL(J:J,1) → 最小値
RANK(値,範囲,0)降順の順位=RANK(J2,J$2:J$1001,0)
RANK.EQ同順位あり同値は同じ順位
RANK.AVG平均順位同値は平均順位
=LARGE(J$2:J$1001, A2)
# A列に順位番号(1,2,3…)があるとき、その順位の売上金額

=INDEX(A$2:A$1001, MATCH(LARGE(J$2:J$1001,A2), J$2:J$1001, 0))
# 上記と組み合わせて「その取引のID」を返す
同額データが複数あると MATCH は最初の一致しか返さない。RANK関数も同順が複数出る点に注意。

⑧ テキスト関数

関数用途例
TRIM前後・連続スペース除去=TRIM(A2)
LEFT(文字列,n)先頭n文字=LEFT(A2,3) → "CUS"
RIGHT(文字列,n)末尾n文字=RIGHT(A2,3)
MID(文字列,開始,n)途中n文字=MID(A2,5,3)
LEN文字数=LEN(A2)
SUBSTITUTE文字置換=SUBSTITUTE(A2," ","")
FIND / SEARCH文字位置検索=FIND("-",A2)
TEXT(値,書式)値→書式付き文字列=TEXT(B2,"yyyy/mm")
CONCAT / &文字列結合=A2&"-"&B2
UPPER / LOWER大文字・小文字変換=LOWER(A2)
=LEFT(A2, FIND("-",A2)-1)
# "TRX-00001" の "-" 前の部分を抽出 → "TRX"

=TEXT(B2,"yyyy") & "年" & TEXT(B2,"m") & "月"
# 日付を "2025年1月" 形式の文字列に

⑨ エラー処理

関数対象エラー例
IFERROR(式, 代替値)すべてのエラー=IFERROR(VLOOKUP(...),"未登録")
IFNA(式, 代替値)#N/A のみ=IFNA(VLOOKUP(...),"")
IF(ISERROR(式), 代替, 式)古いExcel対応IFERROR相当
IFERROR は数式バグも隠すので、最終仕上げまで使わずエラーを確認してから追加する習慣を。
=IFERROR(VLOOKUP(A2,'2_商品マスタ'!$A:$D,2,FALSE), "未登録")
=IFERROR(C2/B2, 0)       # 0除算(#DIV/0!)を0に
=IFNA(XLOOKUP(A2,...), "該当なし")

⑩ 実務 Tips

参照の種類 ― F4キーで切り替え

記法名前コピー時の動き
A2相対参照行・列どちらもズレる
$A$2絶対参照ズレない（マスタ範囲に使う）
$A2列固定の混合行だけズレる（クロス集計の行ヘッダ）
A$2行固定の混合列だけズレる（クロス集計の列ヘッダ）
クロス集計は行ヘッダを $A2、列ヘッダを B$1 にするのが定石。

テーブル化（Ctrl + T）の利点

  • 行追加で数式・書式が自動拡張
  • 構造化参照: =売上データ[売上金額] のように列名で参照できる
  • フィルタ・並び替えが組み込まれる
  • SUMIF等の範囲が自動で伸縮

数式デバッグの手順

  1. F2 で数式編集モード → 参照セルが色分けされ確認できる
  2. 数式バーで部分選択 → F9 で部分評価（ESCで戻ること）
  3. [数式] タブ → [数式の評価] で段階実行
  4. SUMIFS が0になるとき → 条件の型（数値/文字列）を確認

よくあるミス

症状原因対処
SUMIFS が 0 を返す数値列に文字列が混入=VALUE() で変換 / データ確認
VLOOKUP が #N/A前後スペース、全角半角の違い=TRIM() 後に検索
日付でSUMIFSが効かない日付をテキストで持っている=DATEVALUE() で変換
COUNTIF が想定より多いワイルドカード(*)の意図せぬ一致完全一致の場合は ~* でエスケープ

⑪ よく使うパターン集

月別集計テーブル（縦型）

# A列に月番号(1〜12)を入力、B列に以下を入力してコピー
=SUMPRODUCT((MONTH('1_売上データ'!$B$2:$B$1001)=A2)*'1_売上データ'!$J$2:$J$1001)

# 前月比
=IFERROR(B3/B2-1, "")   # 書式をパーセント表示に

地域×カテゴリ クロス集計

# A列に地域名、1行目にカテゴリ名を置き、B2に入力して全体にコピー
=SUMIFS('1_売上データ'!$J:$J,
        '1_売上データ'!$E:$E, $A2,
        '1_売上データ'!$F:$F, B$1)
# $A2 = 行ヘッダ（地域）、B$1 = 列ヘッダ（カテゴリ）

ランキング表（取引ID + 金額）

# A列に順位(1〜10)、B列に金額、C列に取引ID
B2: =LARGE('1_売上データ'!$J:$J, A2)
C2: =INDEX('1_売上データ'!$A:$A, MATCH(B2,'1_売上データ'!$J:$J,0))

顧客セグメント分析

# 顧客別の購入回数・総額・平均をCOUNTIF/SUMIFで集計後、
# PERCENTILE でセグメント境界を算出
=PERCENTILE(購入回数範囲, 0.8)   # 上位20%の境界値

# セグメント割り当て（IFS関数: Excel2019以降）
=IFS(C2>=高頻度境界,"ヘビー", C2>=中頻度境界,"ミドル", TRUE,"ライト")

条件付き件数の割合

=COUNTIF(E:E,"東京") / COUNTA(E2:E1001)
# 東京の割合（書式をパーセントに）

=COUNTIFS(D:D,"ゴールド",E:E,"東京") / COUNTIF(D:D,"ゴールド")
# ゴールド会員の中で東京の割合

外れ値チェック（IQR法）

Q1  = PERCENTILE(J2:J1001, 0.25)
Q3  = PERCENTILE(J2:J1001, 0.75)
IQR = Q3 - Q1
上限 = Q3 + 1.5 * IQR
下限 = Q1 - 1.5 * IQR

# 外れ値の件数
=COUNTIF(J2:J1001,">"&上限セル) + COUNTIFS(J2:J1001,"<"&下限セル, J2:J1001,"<>0")
