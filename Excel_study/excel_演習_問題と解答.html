シート 1_売上

問題 1-1（SUMIFS）

条件: 「担当が田中かつ月が4かつカテゴリが備品」の金額だけを合計する。

  • 解答（例・F2 セル）
    =SUMIFS(D:D,A:A,4,B:B,"田中",C:C,"備品")
    （データが A:D の1行目ヘッダー、2行目からの場合は D2:D100 のように範囲を絞ってもよい）

  • 数式の読み方（引数ごとの解説）

    引数の位置値意味
    第1引数（合計範囲）D:D合計したい数値が入っている列（＝金額列）。条件に合った行の D 列の値だけが足される
    第2引数（条件範囲1）A:A1つ目の条件を調べる列（＝月列）
    第3引数（条件1）4A 列が 4（＝4月）である行だけを対象にする
    第4引数（条件範囲2）B:B2つ目の条件を調べる列（＝担当列）
    第5引数（条件2）"田中"B 列が 田中 である行だけを対象にする
    第6引数（条件範囲3）C:C3つ目の条件を調べる列（＝カテゴリ列）
    第7引数（条件3）"備品"C 列が 備品 である行だけを対象にする

    つまり「A列が4 かつ B列が田中 かつ C列が備品」をすべて満たす行の D 列（金額）だけを合計する、という意味です。

  • 解説
    SUMIFS は 合計する列を最初に書き、その後に「条件範囲1, 条件1, 条件範囲2, 条件2, …」と並べます。条件がすべて満たされた行だけが加算されます。
    構文: =SUMIFS(合計範囲, 条件範囲1, 条件1, [条件範囲2, 条件2], …)

  • 別の書き方

    1. 範囲を明示的に絞る書き方（推奨：大きなシートではパフォーマンスが向上）
      =SUMIFS(D2:D100,A2:A100,4,B2:B100,"田中",C2:C100,"備品")

    2. セル参照で条件を指定する書き方（条件を変えやすい）
      条件をセルに入れておく（例: I2=4, J2="田中", K2="備品"）場合:
      =SUMIFS(D:D,A:A,I2,B:B,J2,C:C,K2)

    3. SUMPRODUCT で同じ結果を得る書き方（古い Excel や条件が複雑なとき）
      =SUMPRODUCT((A2:A100=4)*(B2:B100="田中")*(C2:C100="備品")*D2:D100)
      ※ 各条件を (条件) で囲み * で AND 結合。TRUE=1, FALSE=0 として掛け算で絞り込む仕組み。

    4. FILTER を使う書き方（Excel 365 / Excel 2021 以降）
      =SUM(FILTER(D2:D100,(A2:A100=4)*(B2:B100="田中")*(C2:C100="備品")))
      ※ FILTER で条件に合う行だけを抽出し、SUM で合計。中間結果を確認しやすい利点がある。

問題 1-2（SUMIF）

条件: カテゴリが「消耗品」の金額合計。

  • 解答（例）
    =SUMIF(C:C,"消耗品",D:D)

  • 解説
    条件が1つだけなら SUMIF(範囲, 条件, 合計範囲) で足ります。

問題 1-3（SUMPRODUCT）

条件: 下段の見積表（A13:C15：品目・単価・数量）について、小計列を作らずに合計金額を1セルで求める。

  • 解答（例・F14）
    =SUMPRODUCT(B14:B15,C14:C15)

  • 解説
    単価列と数量列を掛け合わせた配列の合計を一度に出します。行が増えたら範囲を B14:B20 のように広げます。

シート 2_端数_ROUND

データ: A2＝税抜金額、B2＝税率（0.1＝10%）。

問題 2-1（税込・四捨五入）

税込金額を「1円単位で四捨五入」する。

  • 解答（例・D2）
    =ROUND(A2*(1+B2),0)

  • 解説
    ROUND(数値, 0) で小数第1位を四捨五入し、整数円にします。

問題 2-2（切り捨て・切り上げ）

同じ税込額を、それぞれ円未満切り捨て／切り上げで求める。

  • 解答（例）
    • 切り捨て: =ROUNDDOWN(A2*(1+B2),0)
    • 切り上げ: =ROUNDUP(A2*(1+B2),0)
  • 解説
    請求・表示ルールに合わせて、端数の扱いを使い分けます。

シート 3_SUBTOTAL

データ: A2:B6 が部門と金額の明細（小計行なし）。

問題 3-1（SUBTOTAL と SUM の違い）

  1. B8 に =SUBTOTAL(9,B2:B6) を入力。
  2. B9 に =SUM(B2:B6) を入力。
  3. 1行目にオートフィルタをかけ、「部門」で「営業」だけ表示にする。
  4. B8 と B9 の値がどう変わるか比較する。
  • 解答（挙動）
    • SUBTOTAL(9, …) … フィルタで非表示になった行は合計に含めない。
    • SUM … 非表示行も含めたまま合計することが多い（仕様に依存するため、必ず実機で確認）。
  • 解説
    マニュアルのとおり 9 は「合計」です。集計行を表に混在させる場合は、二重計上を避ける設計とセットで使います（本シートは明細のみのシンプル例）。

シート 4_商品マスタ ＋ 5_受注一覧

マスタ: A1:C4（商品コード・商品名・標準単価）。
受注: A2:D6 に「入力コード」（前後スペースあり）、支店、ステータス。

問題 5-1（TRIM）

A列のコードの前後スペースを除いた値を F 列に出す。

  • 解答（例・F2）
    =TRIM(A2)
    下方向へコピー。

  • 解説
    マスタ照合の前に Trim すると、VLOOKUP の不一致が減ります。

問題 5-2（LEFT）

Trim 後のコードの先頭2文字を分類コードとして G 列に出す。

  • 解答（例・G2）
    =LEFT(F2,2)

  • 解説
    先頭 n 文字で大分類する典型的な使い方です。

問題 5-3（VLOOKUP）

Trim した商品コードをキーに、商品名を H 列へ（マスタ全体を第2引数に）。

  • 解答（例・H2）
    =VLOOKUP(F2,'4_商品マスタ'!$A$2:$C$4,2,FALSE)
    シート名が環境で変わる場合は、マスタ範囲をクリックで指定。

  • 解説
    第4引数は 0 または FALSE（完全一致） が鉄則です。列番号は「検索列を1列目としたときの何列目か」です。

問題 5-4（XLOOKUP・Excel 365 等）

同じく商品名を取得。見つからないときは「該当なし」。

  • 解答（例）
    =XLOOKUP(F2,'4_商品マスタ'!$A$2:$A$4,'4_商品マスタ'!$B$2:$B$4,"該当なし")

  • 解説
    列挿入に強く、第4引数で未検索時の表示を指定できます。未搭載の Excel では VLOOKUP + IFERROR で代替します。

問題 5-5（IFERROR）

存在しないコード（例: 手入力で P-9999）を F 列に置いたとき、VLOOKUP の #N/A を空白にする。

  • 解答（例）
    =IFERROR(VLOOKUP(F2,'4_商品マスタ'!$A$2:$C$4,2,FALSE),"")

  • 解説
    一覧表示や帳票でエラー文字を出したくないときに使います。

問題 5-6（COUNTIFS）

「東京支店」かつ「未入金」の件数を1セルで数える。

  • 解答（例）
    =COUNTIFS(B2:B6,"東京",C2:C6,"未入金")

  • 解説
    条件が複数あるときは COUNTIFS。今回のデータ（B2:B6・C2:C6）では 3件 になります。

問題 5-7（SUBSTITUTE・任意）

A2 の文字列から半角スペースをすべて削除する（コード内のスペース対策）。

  • 解答（例）
    =SUBSTITUTE(A2," ","")

  • 解説
    連続スペースや全角スペースが混じる場合は、Trim や複数回の SUBSTITUTE と組み合わせます。

シート 6_日付

A2＝契約開始日、B2＝納期、C2:C3＝祝日の例（2026/3/20, 2026/4/29）。

問題 6-1（WORKDAY）

契約開始日から10営業日後の日付。祝日は C2:C3 を除外。

  • 解答（例・E2）
    =WORKDAY(A2,10,C2:C3)

  • 解説
    第3引数は祝日のセル範囲です。土日は自動で除きます。

問題 6-2（NETWORKDAYS）

A2 から B2 までの営業日数（同じく祝日除外）。

  • 解答（例）
    =NETWORKDAYS(A2,B2,C2:C3)

  • 解説
    開始日・終了日両方を含む日数の数え方になります（Excel の定義どおり実機で確認してください）。

問題 6-3（EOMONTH）

「契約開始日がある月の月末」と「翌月の月末」。

  • 解答（例）
    • 当月月末: =EOMONTH(A2,0)
    • 翌月末: =EOMONTH(A2,1)
  • 解説
    第2引数 0 は当月、1 は1ヶ月後の月末です。

問題 6-4（DATEDIF）

契約開始日から今日までの経過月数（または日数）。

  • 解答（例）
    • 月数: =DATEDIF(A2,TODAY(),"M")
    • 日数: =DATEDIF(A2,TODAY(),"D")
  • 解説
    TODAY() は開いた日によって変わります。第3引数 "M" "D" "Y" などで単位を指定します。

第4章（実務Tips）に相当する演習（ブック外の操作）

マニュアル第4章の内容は、次のように操作だけで演習できます（専用シートはありません）。

問題 T-1（入力規則・プルダウン）

5_受注一覧 の「支店」列に、東京／大阪／名古屋 のみ選べるリストを設定する。

  • 手順の要点
    [データ] → [データの入力規則] → [リスト] → ソースに 東京,大阪,名古屋 またはセル範囲。

  • 解説
    表記ゆれを防ぎ、後続の集計・VLOOKUP のミスを減らします。

問題 T-2（絶対参照・F4）

5_受注一覧 の H2 に VLOOKUP を入れ、コピーしてもマスタ範囲がズレないようにする。

  • 解答の要点
    マスタ範囲を $A$2:$C$4 のように F4 で絶対参照にする。

  • 解説
    行方向にコピーする場合は、検索値だけ相対参照のままにするのが一般的です。

問題 T-3（テーブル化・Ctrl + T）

1_売上 の明細（A1:D8）を選択し Ctrl + T でテーブル化する。

  • 解説
    行追加時に書式や数式の拡張がしやすくなります（Excel のバージョンで挙動が多少異なります）。

正解の確認のしかた

  • 数式の結果は、フィルタや TODAY() により環境で変わるものがあります。
  • 手計算や、別セルに中間結果を置いて、SUM / 件数が一致するか確認すると理解が進みます。

xlsx の再生成

Anything フォルダで、依存関係を置いたうえで次を実行します。

PYTHONPATH="./.pydeps" python3 build_excel_practice.py

openpyxl が無い場合は pip install openpyxl、または pip install --target ./.pydeps openpyxl で .pydeps に入れてください。
